# Tours project

## General info

Tours app is an app made for travelers.
They can see general info for each place and mark them as 'not interested'.

## Link to an app
https://ivanapusic.github.io/tours/

## Technologies

- React 18
- CSS 3
- Vite 4.1.0

## Setup

To run this project, install it using npm

```
$npm install
$npm run dev
```
